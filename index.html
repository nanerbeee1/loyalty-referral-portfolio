About This Project

This project demonstrates my experience supporting Shopify merchants using a loyalty and referral app. It includes a demo of the reward and referral flows, as well as real-world common merchant issues and how I resolve them.

Skills showcased: Shopify platform knowledge, Tier 1 support processes, problem-solving, and basic frontend development.

How It Works Common Issues Support Flow

How It Works

Earn Points

Customers earn points for purchases, sign-ups, and other actions.

Redeem Rewards

Points can be redeemed for discounts, free products, or special offers.

Refer Friends

Customers share referral links and earn rewards when friends place orders.

Common Merchant Issues

Points Not Syncing

Orders are not appearing in the loyalty dashboard.

Support checks order eligibility, app permissions, and webhook status.

Referral Not Tracked

Referral links were clicked but rewards were not applied.

Support verifies referral cookies, order source, and referral program rules.

App Permissions Missing

The app does not have required Shopify permissions.

Merchant is asked to reinstall or update app permissions in Shopify Admin.

Support Resolution Flow

  1. Confirm merchant store and app installation
  2. Check order status and eligibility
  3. Verify app permissions and settings
  4. Escalate if needed
